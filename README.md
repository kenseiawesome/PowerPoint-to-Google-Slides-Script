# powerpoint-to-google-slides-script
Google App Script to change all PowerPoints in a drive folder into Google Slides 

## How to use
- Go to 'script.google.com' and create a new project
- Add Drive API Service V3 to your script
- Paste the code in script.js into your script
- Add the desired folder IDS
- Give script the required permissions/access
- Run the script/function
